# nmea_parcer_c_code
This is a very simple example of parsing NMEA packets

How to use:

git clone https://github.com/khomin/nmea_parcer_c_code.git
cd ./nmea_parcer_c_code
mkdir ./build
cd ./build
cmake ../
make

#start execution
./nmea_parcer_c_code
